# x-leetcode Algorithmic Training

本项目用于系统化学习 LeetCode 算法题，侧重于思维模型推演与 Python 实践。

## 🤖 AI 协作指引 (AI Instruction)

**如果你是初次接入本项目的 AI 助手或语言模型，请务必执行以下指令：**

> “请阅读项目根目录下 `AGENTS.md` 文件，并严格按照其中的教学规范来指导我学习 LeetCode。”

## 项目结构 (Structure)
- `AGENTS.md`：包含 AI 协作规范与指令。
- `learning/`：记录分阶段的算法学习笔记、大纲及思维引导案例。
- `solutions/`：用于存放具体的 Python 解题代码。
