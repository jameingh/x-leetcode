## 🏁 第一阶段：线性探索与空间博弈

### 1. 双指针模式 (Double Pointers)
- **核心逻辑**：利用有序性，通过 `left` 和 `right` 的收缩排除无效搜索空间。
- **代表作**：167. 两数之和 II、11. 盛最多水的容器、142. 环形链表 II。
- **思维沉淀**：建立“边界逼近”直觉。

### 2. 滑动窗口模式 (Sliding Window)
- **核心逻辑**：维护一个“动态手风琴”，通过 `right` 扩张增加内容，`left` 收缩寻找极值。
- **代表作**：3. 无重复字符的最长子串、209. 长度最小的子数组、438. 找到字符串中所有字母异位词。
- **思维沉淀**：掌握了“进窗口 -> 出窗口 -> 更新结果”的标准化节奏。

### 3. 链表技巧 (Linked List)
- **核心逻辑**：掌握“指针手术”，深入理解“哑节点”和“原地反转”。
- **代表作**：206. 反转链表、21. 合并两个有序链表、19. 删除链表的倒数第 N 个节点。
- **思维沉淀**：建立“手术刀”模型，能够精准操控节点引用。

## 🚀 第二阶段：层级搜索与分治思想

### 1. 二分搜索模式 (Binary Search)
- **核心逻辑**：一刀两断，降维打击。
- **实战成果**：
    - [x] **704. 二分查找**：确立了 `while left <= right` 的稳健模版。
    - [x] **35. 搜索插入位置**：领悟了循环结束后 `left` 指针指向“第一个大于等于目标”位置的边界意义。
    - [x] **34. 在排序数组中查找元素的第一个和最后一个位置**：通过“记录并挤压”策略，攻克了 O(log N) 的双边界判定。
- **思维沉淀**：建立 **“两墙挤压”物理模型**。理解了循环退出时 `left` 和 `right` 错位产生的物理边界意义。

---

### 2. 二叉树与递归 (Binary Tree & Recursion)
- **核心逻辑**：将大问题拆解为左右子树的子问题，利用递归“分身”并行探索。
- **实战成果**：
    - [x] **104. 二叉树的最大深度**：确立了“向上汇报”的递归汇总直觉。
    - [x] **226. 翻转二叉树**：掌握了“向下分发”的递归动作与 Python 原子交换语法糖。
    - [x] **101. 对称二叉树**：实现了“双人同步对讲”模式，掌握了同步控制两个递归引用的技巧。
    - [x] **222. 完全二叉树的节点个数**：快速迁移“向上汇报”模型，实现了高效的总数统计。
    - [x] **112. 路径总和**：实现了“携带余额下探”模型，理解了如何由根节点向下传递并消耗状态。
- **思维沉淀**：建立 **“部门分身术”** 模型。理解了递归就是主管下放权力，处理局部后由系统自动回馈汇总。

---

### 3. 二叉搜索树 (Binary Search Tree)
- **核心逻辑**：利用“左小右大”的特性进行单边裁剪，这其实就是把二分搜索融入到了树里面。
- **实战成果**：
    - [x] **700. 二叉搜索树中的搜索**：确立了在 BST 中进行“单边探索（剪枝）”的直觉，摆脱了完整遍历的思维惯性。
    - [x] **98. 验证二叉搜索树**：掌握了“区间约束下发（带着尚方宝剑查岗）”的高维思维，突破了只看父子节点的局限性。
    - [x] **530. 二叉搜索树的最小绝对差**：利用中序遍历的严格递增魔法，配合对 `prev` 的巧妙“判空防 0 保护”，成功将二维的树降维为一维数组问题。
    - [x] **701. 二叉搜索树中的插入操作**：领悟了修改树结构时最核心的 **“重接”思维**（利用 `root.left / right` 接住下层返回的子树，再 `return root`）。
- **思维沉淀**：建立了一套从读、验证、降维到修改极其完善的 BST 知识体系，并提炼出了“不落单的外部状态变量必须判空/边界”的高维内功。

### 4. 广度优先搜索 (BFS)
- **核心逻辑**：利用队列结构，通过“先进先出”的钢铁洪流，模拟无差别的“水波扩散”。
- **实战成果**：
    - [x] **102. 二叉树的层序遍历**：舍弃了递归路线，成功实践了“外层 while 横扫，内层 for 锁死当前人数”的两层循环模型。
- **思维沉淀**：确立了“求最短路径或按层处理，自带代际隔离的 BFS 就是最优解”的思维钢印。
